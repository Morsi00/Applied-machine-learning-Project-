Data Cleaning

Issues identified and solutions applied

Issue 1: Drop customerID

Cleaning

Problem: The customerID column contains no useful information for analysis.

Solution:

data.drop("customerID", axis=1, inplace=True)

Result: Column successfully dropped, reducing noise in the data.

Issue 2: Convert TotalCharges

Type Conversion

Problem: TotalCharges column was stored as object instead of numeric.

Solution:

data['TotalCharges'] = pd.to_numeric(data.TotalCharges, errors='coerce')

Result: Column converted to numeric values with error handling.

Issue 3: Missing Values

Missing Values

Problem: NaN values found in TotalCharges column.

Solution:

data.isnull().sum()
data[np.isnan(data['TotalCharges'])]

Result: 11 rows identified with missing values — all new customers with Tenure = 0.

Issue 4: Encode Categorical Data

Encoding

Problem: Categorical features need to be converted to numeric for models.

Solution:

from sklearn.preprocessing import LabelEncoder
le = LabelEncoder()
for col in categorical_cols:
    data[col] = le.fit_transform(data[col])

Result: All categorical columns successfully encoded as numeric values.

Issue 5: Feature Scaling

Advanced Processing

Problem: Features have very different ranges (e.g. Tenure: 0–72, MonthlyCharges: 18–118).

Solution:

from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
data_scaled = scaler.fit_transform(data)

Result: All features normalized to improve model performance.
