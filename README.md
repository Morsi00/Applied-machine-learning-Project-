## Key Insights & Conclusion

The notebook highlights the following key insights and conclusions from the analysis:

*   **Cluster 3 is a top priority for the retention team**: Approximately 4-5 out of every 10 customers in this cluster are likely to churn.
*   **Cluster 1 represents VIP customers**: These customers exhibit a high anomaly rate, suggesting they require special treatment and maintenance to prevent churn.
*   **Optimal Threshold = p90**: This threshold covers 704 customers with a 16.5% churn rate, making it more actionable for business strategies than p95.
*   **Autoencoder Success**: The Autoencoder model successfully performed two simultaneous tasks:
    *   **Anomaly Detection**: Identified "Premium Customers" who need special attention, not necessarily because they are likely to churn, but because their behavior deviates significantly, indicating a potentially high cost if they leave.
    *   **Clustering**: Divided customers into 4 clear segments based on learned representations rather than raw features, providing deeper segmentation than standard K-Means.

## How to Run the Code

To run this project, you will need a Google Colab environment or a local Python environment with Jupyter Notebook installed. The main steps are as follows:

1.  **Open the Notebook**: Access the Google Colab notebook via the provided link: [https://colab.research.google.com/drive/1bx8m5J-3dr4zPfxWxTWx712EFflx1dwG?usp=sharing](https://colab.research.google.com/drive/1bx8m5J-3dr4zPfxWxTWx712EFflx1dwG?usp=sharing)

    ```bash
    pip install kagglehub pandas numpy
    ```
3.  **Run Cells Sequentially**: Execute each cell in the notebook from top to bottom. The notebook will handle:
    *   Downloading the dataset.
    *   Loading and initial inspection of the data.
    *   Data cleaning and preprocessing.
    *   Model application (Autoencoder, Clustering, Anomaly Detection).
    *   Displaying insights and conclusions.
